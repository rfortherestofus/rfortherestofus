# Convert PDF to Animated GIF

Convert PDF to Animated GIF

## Usage

``` r
pdf_to_gif(
  pdf_location,
  gif_file_path = here::here(),
  gif_file_name,
  density = 50
)
```

## Arguments

- pdf_location:

  Location of PDF file

- gif_file_path:

  Starts at here::here()

- gif_file_name:

  You must include .gif at end

- density:

  Affects file size of GIF that is outputted

## Value

GIF
